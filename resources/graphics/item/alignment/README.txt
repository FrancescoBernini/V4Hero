What are .spr files?
.spr files are used to guide the game on where is the center of given object (like spear, or helm).

What is the center?
Center is supposed to be the point in equipment where it either stays on Patapon's hand or head.
Points are defined by X and Y position in the texture.

There is 1 line, defines a center for Ultra quality setting (3840 x 2160 canvas).

.spr file should be called the exact same way as the item's ID.
Items without a .spr file when equipped, will be placed at 0x0 position of the texture.
